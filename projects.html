My Projects

Python projects

- WOLT and GLOVO Parser: This project provides seamless API integration for virtually any application related to restaurants and the food service industry. It allows developers to retrieve comprehensive information about restaurants, including menus, through API requests. This parser enhances the capabilities of restaurant-related projects by offering up-to-date data and easy integration.

- Google Maps Parser: An API project designed to fetch images and reviews of specific locations from Google Maps upon request. This tool enables applications to incorporate rich, location-based content, improving user experience by providing insightful information about places of interest.

- Steam Skins AutoTrader: A complex system that enables fully automated trading of Steam skins. By leveraging the Steam API, this project automates the buying and selling process, allowing users to engage in the Steam marketplace efficiently without manual intervention.

Game development

- LSpeed MTA Server. I created and developed an MTA (Multi Theft Auto) server that operated successfully from 2019 to 2022. This project showcased my skills in server management, gameplay design, and maintaining a vibrant player community over several years.

- LSpeedV GTA5 Roleplay. An ambitious GTA5 Roleplay project that I worked on for over a year. Though unfinished, the project demonstrated my dedication and ability to manage large-scale development. It was later sold, reflecting its potential and value in the gaming community.

3D Modelling

I specialized in converting and enhancing 3D car models for popular games such as Assetto Corsa, GTA V, and GTA San Andreas. Leveraging advanced tools like Blender and 3ds Max, I meticulously optimized and refined models to ensure exceptional quality and seamless integration into the gaming environments.

This work allowed me to combine technical precision with creative artistry, delivering visually stunning and highly detailed vehicle models that elevate the gaming experience.
